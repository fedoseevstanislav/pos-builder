# Use Case Checklist — Situation Triggers

Internal reference for the diagnostic LLM. NOT shown to the learner.
The LLM reads this to know what to ask about when filling inventory gaps.

## How to use

After the learner's open-ended answer (Phase 3, Step 3.1), compare what they mentioned against this checklist. For uncovered zones, ask the situation-trigger question. Ask through life situations, never through use case names or technical terms.
